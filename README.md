# Ricoma Coding Challenge ...

## Installation

Clone this repo and run Docker command.

```
docker-compose up -d
```

```
http://localhost:3000 | API Url
http://localhost:3001 | API Client Url
http://localhost:3002 | API FrontEnd URL
```
